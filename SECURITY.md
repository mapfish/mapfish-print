# Security Policy

## Supported Versions

| Version | Supported Until |
| ------- | --------------- |
| <= 3.26 | Unsupported     |
| 3.28    | 23/06/2026      |
| 3.29    | Unsupported     |
| 3.30    | Best effort     |
| 3.31    | Best effort     |
| 3.32    | Unsupported     |
| 3.33    | 31/08/2026      |
| 4.0     | Best effort     |

## Reporting a Vulnerability

Please do not open a public GitHub issue to report a vulnerability. Instead, create a [new security advisory](https://github.com/mapfish/mapfish-print/security/advisories/new).
