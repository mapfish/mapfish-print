Printing a WMS Layer

This example prints two wms layer using GetMap Version 1.1.1.

  • One Layer is USA-Population which is in EPSG:4326
  • The other layer is Spearfish Roads with is in EPSG:26713

Click here to see the config.yaml

The js is not minified so it is readable. See print-page.js.

The json that will be posted will look something like: 

                {
   "units":"degrees",
   "srs":"EPSG:4326",
   "layout":"A4 portrait",
   "dpi":56,
   "mapTitle":"Printing Demo",
   "comment":"This is a simple map printed from GeoExt.",
   "layers":[
      {
         "baseURL":"http://localhost:9876/e2egeoserver/wms",
         "opacity":0.5,
         "singleTile":true,
         "type":"WMS",
         "layers":[
            "topp:states"
         ],
         "format":"image/png",
         "styles":[
            ""
         ],
         "customParams":{
            "TRANSPARENT":"true"
         }
      },
      {
         "baseURL":"http://localhost:9876/e2egeoserver/wms",
         "opacity":1,
         "singleTile":true,
         "type":"WMS",
         "layers":[
            "sf:roads"
         ],
         "format":"image/png",
         "styles":[
            ""
         ]
      }
   ],
   "pages":[
      {
         "center":[
            -104.00000000001,
            44.7
         ],
         "scale":4000000,
         "rotation":0
      }
   ]
}
